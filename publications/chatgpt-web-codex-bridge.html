ChatGPT 网页到 Codex 的 Responses API 实验桥接器

3099404236 · 2026-07

摘要

这是一个面向短期实验的 Windows 本地桥接器。它接收 Codex 的 Responses API 请求，通过可见浏览器复用已登录的 ChatGPT 网页对话，再把网页回答转换为文本、函数调用或原生 apply_patch 事件。项目重点处理同一任务的对话复用、工具输出续轮、SSE 事件、错误重试、补丁后备路径和敏感登录数据隔离；它不绕过验证码、用量限制或网站风控。
